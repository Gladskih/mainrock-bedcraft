# Code of Conduct

## Our Standards

We value respectful, constructive collaboration.

Expected behavior:

- be professional and factual
- discuss ideas, not people
- give actionable feedback
- assume good intent and ask clarifying questions

Unacceptable behavior:

- harassment, insults, or personal attacks
- discrimination of any kind
- doxxing or sharing private information
- disruptive trolling or bad-faith participation

## Enforcement

Project maintainers may remove, edit, or reject contributions and comments that violate this code.

For serious concerns, use private maintainer contact or GitHub security reporting channels.
