# Support

## Getting Help

For usage questions, open a GitHub issue with:

- command you ran
- full structured log output
- OS and Node.js version
- reproduction steps

## Bug Reports

Please include:

- expected behavior
- actual behavior
- minimal reproducible setup
- whether the issue is `nethernet` or `raknet`

## Security Issues

Do not post sensitive vulnerabilities publicly.

Follow `SECURITY.md`.

## Scope

This project is currently MVP/beta. Interfaces and behavior may change between minor revisions.

Known limitation:

- RakNet public-server `join` is not a validated compatibility target yet.
